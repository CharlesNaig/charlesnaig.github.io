üîÑ Auto-Management Features

‚è≠Ô∏è

Auto-Skip Absent Users

If a user is not in a voice channel when it's their turn, they are automatically skipped. The system moves to the next person and notifies the channel.

‚è∞

Auto-Close Empty Queue

If the queue becomes empty for 30 minutes, it automatically closes. A notification is sent when this happens.

üîí

Voice Channel Enforcement

Users are automatically removed from queue if they leave the voice channel. Users must be in a voice channel to join.

üÜò Troubleshooting

Common Karaoke Issues

"You must be in a voice channel"

Solution: Join any voice channel in the server before using /joinqueue

"Queue hasn't been started yet"

Solution: A moderator needs to use /startqueue before progression commands work

"You don't have permission"

Solution: Check if you have the required moderator roles or "Manage Channels" permission

Common Artist Profile Issues

"You need the Artist role"

Solution: Contact an admin to get the Artist role or have them create your profile

"Profile data expired"

Solution: The creation session timed out - start over with /artist create

"Invalid URL" errors

Solution: Ensure all URLs start with http:// or https://
